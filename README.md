Este projeto é uma simulação de uma lâmpada feita com HTML, CSS e JavaScript puro (Vanilla JS).
Eu desenvolvi seguindo um vídeo no YouTube, onde aprendi a trabalhar com eventos e manipulação do DOM.

No HTML, criei a estrutura básica da página, com um título, uma imagem que representa a lâmpada e um botão para ligar e desligar.
No CSS, estilizei a página com tema escuro, centralizando os elementos e adicionando efeitos visuais como sombra, transições e animações ao passar o mouse.
No JavaScript, implementei a lógica de funcionamento da lâmpada.

O código permite:

Ligar e desligar a lâmpada ao clicar no botão

Ligar ao passar o mouse sobre a imagem

Desligar ao tirar o mouse da imagem

Quebrar a lâmpada com duplo clique, impedindo que ela ligue novamente

Esse projeto me ajudou a entender melhor como usar eventos, funções, condições e como alterar imagens e textos dinamicamente usando JavaScript, além de reforçar a integração entre HTML, CSS e JS.
